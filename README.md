# 💰 Expense Tracker (Personal Finance CLI Tool)

C# ve Entity Framework Core kullanılarak geliştirilmiş, SQLite veritabanı destekli kişisel harcama ve bütçe takip uygulaması.

## 🛠️ Teknolojiler

- **Dil:** C# (.NET 9)
- **Veritabanı:** SQLite
- **ORM:** Entity Framework Core 9.0

## 🚀 Özellikler

- Harcama Ekleme (Açıklama, Tutar, Kategori, Tarih)
- Tüm Harcamaları ve Toplam Bütçeyi Listeleme
- Otomatik SQLite Veritabanı Oluşturma (EnsureCreated)

## 💻 Kurulum ve Çalıştırma

Projoyi klonladıktan sonra terminalde şu komutu çalıştırmanız yeterlidir:

```bash
dotnet run
```
